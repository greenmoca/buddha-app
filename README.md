# 지금, 붓다라면 — 현대 사회를 부처님의 시선으로 비추는 앱

뉴스 기사나 고민 상황을 입력하면 불교적 관점으로 분석하고, 실제 경전 구절과 실천 방법을 제시하는 웹앱입니다. **API 키 없이, 인터넷 없이도 동작**하며, 폰에 앱처럼 설치할 수 있습니다.

고려시대 감지금니(紺紙金泥) 사경 — 남빛 종이에 금물로 쓴 불경 — 에서 디자인을 가져왔습니다.

---

## 한눈에 보는 핵심 기능

| 기능 | 설명 |
|---|---|
| **지혜 답변** | 질문 속 단어를 분석해 11개 주제(분노·비교·불안·상실·탐욕·번아웃·혐오·기후·외로움·거짓·기본) 중 알맞은 분석과 경전 구절 제시 |
| **복합 답변** | 두 괴로움이 겹친 질문(예: 직장 갈등+번아웃)은 두 주제의 지혜를 함께 제시 |
| **삼현 비교** | 같은 상황을 붓다(법구경)·공자(논어)·노자(도덕경) 관점으로 나란히 비교 |
| **오늘의 법문** | 매일 다른 경전 구절. 부처님오신날·설날·추석은 **음력 자동 계산**으로 전용 법문 표시 |
| **카드 공유** | 답변 속 경전 구절을 1080×1350 이미지 카드로 — 폰에서는 공유 시트로 바로 전송 |
| **위젯 모드** | 주소 뒤 `?view=daily` 를 붙이면 오늘의 법문만 전체 화면으로 |
| **다크/라이트** | 기기 설정 따라 자동 전환 — 밤은 감지금니, 낮은 한지묵서 |
| **AI 분석(선택)** | Anthropic API 키를 입력하면 맞춤 AI 분석으로 업그레이드. 실패 시 내장 사전으로 자동 전환 |

## 파일 구성

```
index.html            앱 전체 (화면 + 지혜 사전 + 엔진이 모두 이 한 파일에)
manifest.webmanifest  앱 정보 (이름·아이콘·바로가기)
sw.js                 서비스 워커 (오프라인 캐시) ★ 수정 배포 시 버전 숫자 올리기
icon-192.png / icon-512.png  앱 아이콘
```

## 빠른 시작

1. **그냥 써보기**: `index.html`을 브라우저로 열면 끝.
2. **앱으로 설치하기 (데스크탑·모바일)**: 배포된 주소를 열면 우측 상단 **"⬇ 앱 설치"** 버튼이 보입니다.
   - **데스크탑(크롬·엣지)**: 버튼을 누르거나 주소창의 설치 아이콘을 클릭 → 창 없는 독립 앱으로 실행됩니다.
   - **안드로이드(크롬·삼성인터넷)**: 버튼을 누르면 설치 팝업이 뜹니다.
   - **iOS(사파리)**: 버튼을 누르면 안내가 나옵니다 — 공유 → "홈 화면에 추가".
   - 설치 후에는 비행기 모드에서도 동작하며, 새 버전이 배포되면 하단에 **"새 버전이 준비됐어요"** 알림이 떠 동의 시에만 갱신됩니다.

## 내 마음대로 고치기 (index.html에서 검색)

- **주제 추가**: `WISDOM_DB` — 기존 블록 하나를 복사해 `default` 앞에 붙이고 내용 수정
- **인식 단어 추가**: `SYNONYMS` — 배열에 단어만 추가하면 끝. 자주 헷갈리는 단어는 각 주제의 `strong`으로 (가중치 3배)
- **법문 늘리기**: `DAILY_VERSES` — 365개를 넣으면 1년 내내 매일 다른 구절 (동봉된 법문-변환기.html 활용)
- **특별한 날 지정**: `SPECIAL_BY_DATE`에 `'YYYY-MM-DD': '키'` 한 줄 (자동 음력 계산보다 우선 적용)
- **색 바꾸기**: `:root`(다크) / `prefers-color-scheme: light` 안의 `:root`(라이트) 변수 8개

## 수정 배포 규칙 (중요)

1. 파일 수정 → **sw.js의 캐시 이름 숫자 올리기** (`buddha-v7` → `buddha-v8`)
2. GitHub에 커밋 → 폰에 설치된 앱은 보통 **두 번째 실행**부터 새 버전 반영

## 매칭 엔진 원리 (요약)

질문을 어절로 쪼개 조사(이/가/을/를…)를 떼고 → 주제별 키워드와 대조 (핵심 단어 3점, 일반 1점, 반복 시 최대 3회 누적) → 키워드 수로 보정한 점수 순위표 작성 → 1위 주제로 답하되, 2위가 충분히 강하면 복합 답변. '영화가'가 '화가'로 오인되지 않도록 짧은 키워드는 어절 단위로만 인정합니다.
